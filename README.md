# 📊 Sorting Visualizer

An interactive web application to visualize how different sorting algorithms work step-by-step.

---

## 🚀 Live Demo
https://gowripranavig.github.io/sorting-visualizer/

---

## 📌 Features

- Visualizes sorting algorithms using bars  
- Start, Pause, Next and Previous controls  
- Adjustable speed  
- Adjustable array size  
- Multiple sorting algorithms  
- Color-coded visualization  

---

## 🧮 Algorithms Implemented

- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Quick Sort  
- Merge Sort  
- Heap Sort  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ How to Run

1. Clone the repository  
2. Open index.html in browser
